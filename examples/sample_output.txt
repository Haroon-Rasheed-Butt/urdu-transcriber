# Example Output - Urdu Meeting Transcription

This folder shows what you can expect from the transcriber.

## Example Workflow

### Input
A 30-minute Urdu team meeting recording: `team_meeting.mp3`

### Command
```bash
python transcribe.py team_meeting.mp3 --clean-audio
```

## Processing Stats

- **Audio Duration:** 30 minutes
- **Processing Time:** ~45 minutes (on CPU)
- **Transcript Length:** ~2,400 words (Urdu)
- **Accuracy:** ~75% (typical for Urdu with clear audio)
- **File Size:** Original 28 MB → Cleaned 12 MB WAV

## Tips for Best Results

1. **Clear Audio:** Background noise should be minimal
2. **Single Language:** Works best when meeting is entirely in Urdu
3. **Review Before Claude:** Check Urdu transcript for obvious errors
4. **Customize Prompt:** Edit claude_prompt.txt for specific needs

## Common Use Cases

### Daily Standup
```bash
python transcribe.py standup.mp3 --no-clean
```
Fast processing, skip noise reduction for short meetings

### Important Client Meeting
```bash
python transcribe.py client_meeting.mp3 --clean-audio --beam-size 10
```
Maximum quality settings

### Batch Weekly Meetings
```bash
python batch_transcribe.py week1.mp3 week2.mp3 week3.mp3
```
Process multiple meetings at once
